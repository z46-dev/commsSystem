# Setup
1. Create a `.env` file with the following configuration:
```js
HOST=localhost
PORT=4096
LOGINS=username1|password1,username2|password2
PRIMEN_X=14
PRIMEN_Y=28
INBOUND_SEED=3490
OUTBOUND_SEED=12839
DATA_FLAGS=location,weather,system
RUN_SERVER=true
RUN_CLIENT=false
SEPARATE_SERVERS=false
RUN_WEBSITE=false
WEBSITE_PORT=8080
WEBSITE_ACCESS_PASSWORD=a_secure_password
USERNAME=username1
PASSWORD=password1
```
